# Screenshots (Company panel)

Yahan Company panel ke har page ka live UI screenshot rakha jata hai. Abhi yeh **placeholder** folder hai.

## Kaise add karein
1. Live site (`http://localhost:4200/company/...` ya production URL) par jaakar relevant page kholein.
2. Full-page screenshot lein.
3. Image ka naam doc me likhe gaye naam jaisa rakhein (e.g. `dashboard.png`, `company-lenden.png`, `banners.png`).
4. Image ko isi `screenshots/` folder me (ya parent folder ke andar wale `screenshots/` me) save karein.
5. Doc file me `![...](screenshots/...png)` line already maujood hai — image aate hi automatically dikh jayegi.

> Nested folders (jaise `live-matches/`, `manage/`) ke andar bhi apna `screenshots/` folder ho sakta hai.
